# Mergio Audio Player

Простой аудиоплеер с десктопным и веб-клиентом в одном монорепо.

## Структура

- `apps/desktop` — десктопное приложение
- `apps/frontend` — веб-клиент

## Стек

- Bun
- TypeScript
- Biome (линтинг и форматирование)

## Установка

```bash
bun install
```

## Запуск

```bash
bun run dev
```
